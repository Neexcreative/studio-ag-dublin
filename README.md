# NEEX Design System

Base do design system da NEEX Creative.

## Estrutura
- Foundations: tokens de cor, tipografia, spacing, radius e shadows
- Components: Button, Card, Badge, Input, Select, Textarea, Container, Section, SectionTitle
- Patterns: Hero, ContactForm
- Templates: LandingPage
- Docs: contexto e prompts para uso com IA

## Stack
- React
- TypeScript
- Vite
- Storybook

## Rodando localmente

```bash
npm install
npm run dev
npm run storybook
