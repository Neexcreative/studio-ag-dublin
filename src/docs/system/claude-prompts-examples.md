# Example Prompts

## Generic Landing Page

Create a landing page using the NEEX Design System for a business that wants to generate leads online.

Requirements:
- use `LandingPage`
- build the page from typed blocks
- include hero, highlights, offer section, CTA and contact form
- keep the copy generic enough to adapt later

## Local Service Business

Create a landing page for a local home service company using the NEEX Design System.

Requirements:
- emphasize trust and speed
- use structured card items for benefits and services
- create a contact form oriented to quote requests

## Personal Brand

Create a landing page for a consultant or strategist using the NEEX Design System.

Requirements:
- emphasize authority and clarity
- use the services section as offer packages
- use the contact form for discovery calls

## Restaurant or Hospitality

Create a landing page for a hospitality business using the NEEX Design System.

Requirements:
- use the hero to present the experience
- use card sections for menu highlights or venue benefits
- use the CTA for reservations or inquiries

## Digital Offer

Create a landing page for a digital product using the NEEX Design System.

Requirements:
- use highlights to explain the value
- use services section as modules, plans or outcomes
- use CTA and contact blocks to support conversion
